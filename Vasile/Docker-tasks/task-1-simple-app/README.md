# Task 1: Dockerized Flask Application

## Project Structure
- `app.py`: Flask application with `/`, `/health`, and `/info` endpoints.
- `Dockerfile`: Optimized multi-layer build using a non-root user.
- `.dockerignore`: Excludes local environments and git history.
- `test-commands.sh`: Automation script to verify endpoints.
